Contact Force Estimation

E08 Advanced Topics Force Estimation Planar Leg Est. 2003

External contact forces can be inferred from the discrepancy between predicted and observed motion via the momentum observer. When an unexpected force acts on the robot, the generalized momentum deviates from its model prediction, and this residual directly reveals the disturbance.

Overview

In many robotic applications—from collaborative manipulation to legged locomotion—the robot must detect and quantify external contact forces without a dedicated force/torque sensor. The momentum observer achieves this by monitoring the generalized momentum $\mathbf{p} = \mathbf{M}(\mathbf{q})\dot{\mathbf{q}}$ and comparing its rate of change to the known applied torques.

Any mismatch between the expected momentum evolution (from gravity, Coriolis, and commanded torques) and the actual measured momentum is attributed to an external disturbance. This residual signal converges to the true external force with a first-order lag whose bandwidth is set by the observer gain $K_O$.

Key Insight

The momentum observer needs only joint position and velocity measurements—no acceleration or force sensor. It exploits the passivity of the rigid-body dynamics: all energy injected by an external force must appear as a momentum change beyond what the model predicts.

Collision Detection Pipeline

  • Phase 1 — Detection: The residual $r(t)$ exceeds a threshold, signalling contact.
  • Phase 2 — Isolation: Identify which joint or link is in contact (multi-DOF case).
  • Phase 3 — Identification: Estimate the magnitude and direction of the external force.
  • Reaction: Trigger a safe reflex, impedance response, or controlled retreat.

Key Properties

  • Sensorless — No force/torque sensor required at the contact point
  • First-order lag — Bandwidth controlled by $K_O$; larger gain = faster response but more noise
  • Model-dependent — Accuracy relies on a good dynamic model (inertia, gravity, friction)
  • Passivity-based — Leverages energy/momentum balance of rigid-body dynamics

How It Works

  1. Compute model torques: From the current joint state $(\mathbf{q}, \dot{\mathbf{q}})$, compute the expected torques from gravity, Coriolis, and damping: $\boldsymbol{\tau}_{\text{model}}$.
  2. Measure momentum: Calculate $\mathbf{p} = \mathbf{M}(\mathbf{q})\dot{\mathbf{q}}$ from joint velocity and the inertia matrix.
  3. Integrate the residual: The observer maintains an internal integral: $\boldsymbol{\beta}(t) = \int_0^t \left[\boldsymbol{\tau}_{\text{model}} + \mathbf{r}(s)\right] ds$.
  4. Compute residual: $\mathbf{r}(t) = K_O \left[\mathbf{p}(t) - \mathbf{p}(0) - \boldsymbol{\beta}(t)\right]$.
  5. Interpret: The residual $\mathbf{r}(t)$ converges to the true external torque $\boldsymbol{\tau}_{\text{ext}}$ with time constant $1/K_O$.
Gain Trade-off

Increasing $K_O$ makes the observer respond faster to contacts, but it also amplifies measurement noise. In practice, $K_O$ is tuned to achieve a detection delay of 5–20 ms while keeping false-alarm rates low.

Residual Filtering

The first-order observer structure acts as a low-pass filter on the external force with cutoff frequency $\omega_c = K_O$. For a step contact force $f_0$ applied at $t_c$:

$$\hat{f}(t) = f_0 \left(1 - e^{-K_O(t - t_c)}\right), \quad t \geq t_c$$

This means the estimated force ramps up exponentially toward the true value. The 95% settling time is approximately $3/K_O$ seconds.

Key Equations

Rigid-Body Dynamics

$$\mathbf{M}(\mathbf{q})\ddot{\mathbf{q}} + \mathbf{C}(\mathbf{q},\dot{\mathbf{q}})\dot{\mathbf{q}} + \mathbf{g}(\mathbf{q}) = \boldsymbol{\tau} + \boldsymbol{\tau}_{\text{ext}}$$

Generalized Momentum

$$\mathbf{p} = \mathbf{M}(\mathbf{q})\dot{\mathbf{q}}$$

Momentum Rate (Key Identity)

$$\dot{\mathbf{p}} = \boldsymbol{\tau} + \boldsymbol{\tau}_{\text{ext}} + \left(\dot{\mathbf{M}} - \mathbf{C}\right)\dot{\mathbf{q}} - \mathbf{g}$$

Using the skew-symmetry property $\dot{\mathbf{M}} - 2\mathbf{C}$ is skew-symmetric, the term $(\dot{\mathbf{M}} - \mathbf{C})\dot{\mathbf{q}}$ depends only on known quantities.

Momentum Observer Residual

$$\mathbf{r}(t) = K_O \left[\mathbf{p}(t) - \mathbf{p}(0) - \int_0^t \left(\boldsymbol{\tau}_{\text{known}} + \mathbf{r}(s)\right) ds \right]$$

where $\boldsymbol{\tau}_{\text{known}} = \boldsymbol{\tau} + (\dot{\mathbf{M}} - \mathbf{C})\dot{\mathbf{q}} - \mathbf{g}$ collects all model-based terms.

Residual Dynamics

$$\dot{\mathbf{r}} = K_O \left(\boldsymbol{\tau}_{\text{ext}} - \mathbf{r}\right)$$

This is a stable first-order system: $\mathbf{r}(t) \to \boldsymbol{\tau}_{\text{ext}}$ with time constant $1/K_O$.

1-DOF Simplification (this demo)

$$p = I\dot{q}, \quad \dot{p} = \tau_g + \tau_{\text{ext}} - b\dot{q}$$

$$r(t) = K_O\left[p(t) - p(0) - \int_0^t\left(\tau_g + \tau_{\text{damp}} + r\right)ds\right]$$

$$\hat{f}_{\text{ext}} = r / L$$

MATLAB Implementation

MATLAB
%% E08: Momentum Observer — Contact Force Estimation
% 1-DOF planar leg with step contact force

% Parameters
m = 2.0; L = 0.5; g = 9.81; b = 0.3;
I = (1/3)*m*L^2;            % rod inertia about pivot
K_O = 30;                    % observer gain
dt = 0.005; T = 4.0;
t = 0:dt:T;  N = length(t);

% Contact force profile (step at t_contact)
t_contact = 1.5;  f_mag = 5.0;
f_ext = f_mag * (t >= t_contact);  % external force (N)

% Preallocate
q = zeros(1,N); qd = zeros(1,N);
r = zeros(1,N);                     % residual
beta = 0;                           % integral accumulator

q(1) = 0.3; qd(1) = 0;

for k = 1:N-1
    % True dynamics (RK2)
    tau_g = -m*g*(L/2)*sin(q(k));
    tau_ext = f_ext(k)*L;
    qdd1 = (tau_g + tau_ext - b*qd(k)) / I;

    q_mid  = q(k)  + qd(k)*dt;
    qd_mid = qd(k) + qdd1*dt;
    tau_g2 = -m*g*(L/2)*sin(q_mid);
    tau_ext2 = f_ext(min(k+1,N))*L;
    qdd2 = (tau_g2 + tau_ext2 - b*qd_mid) / I;

    q(k+1)  = q(k)  + 0.5*(qd(k)+qd_mid)*dt;
    qd(k+1) = qd(k) + 0.5*(qdd1+qdd2)*dt;

    % --- Momentum Observer ---
    p_meas = I * qd(k+1);              % measured momentum
    tau_model = -m*g*(L/2)*sin(q(k+1)) - b*qd(k+1);
    beta = beta + (tau_model + r(k))*dt;
    r(k+1) = K_O * (p_meas - I*qd(1) - beta);
end

% Convert residual to force estimate
f_hat = r / L;

% Plot
figure;
subplot(2,1,1); hold on;
plot(t, f_ext, 'r-', 'LineWidth', 2);
plot(t, f_hat, 'b--', 'LineWidth', 2);
legend('True f_{ext}', 'Estimated f_{ext}');
xlabel('Time (s)'); ylabel('Force (N)');
title('Momentum Observer: Contact Force Estimation');

subplot(2,1,2);
plot(t, q, 'g-', 'LineWidth', 2);
xlabel('Time (s)'); ylabel('Joint angle (rad)');
title('Joint Motion');
